Now you begin

As you begin building your website, one of the most important things you can do is to use lots of description in what elements are being included in the site. Don't forget that div elements can be your best friend when trying to set a class. And the span element is so important for when you want a section of text to have its own formatting. You can use it to set a class without disturbing the flow of the text.

How to Table
Helpful Elements How to Use Them
<span> Used within other elements Easy reference in CSS Very necessary
<thead> Mark header in table Keep your header separate Also good for CSS reference
<a> Anchor a link Used with <href="url"> Also used to jump to sections in page
<img src"url"> Used to insert images Can be used from local directory or URL Can also be used with anchor tag for clickable images
<div> Used to divide page into sections When used with anchor tag, you can jump to that section Add href=#id in order to link the section
<ol> This creates a numbered list Very helpful for showing steps <ul> can be used for bullet points
